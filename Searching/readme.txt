Для запуска программы нужно запустить файл start.bat

Работа программы:
1) Указать папку для поиска, нажав соответствующую кнопку.
2) В поле "Текст" ввести искомое слово
3) Затем выбрать расширение файла (по умолчанию "log")
4) Нажав на кнопку "Поиск" отобразится список удовлетворяющих поиску файлов
5) Для отображения содержимого какого-либо файла, его нужно выделить, нажав по нему левой кнопкой мыши
6) С помощью управляющих кнопок можно выделить все совпадения искомого текста, снимать их и переходить по совпадениям вперед/назад и 

P.S. jar-файл с приложением находится в папке с проектом, открыв который можно увидеть исходный код для проверки
